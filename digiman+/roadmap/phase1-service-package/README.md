# Digiman+ Roadmap

Folder ini menyimpan diskusi, analisis, dan dokumen perencanaan terkait arah pengembangan Digiman+ ke depan.

## Struktur

| Topik | Deskripsi |
|-------|-----------|
| `feature-discovery/` | Ide dan eksplorasi fitur baru |
| `ux-journey/` | User journey dan UX improvement |
| `technical-direction/` | Arah arsitektur dan teknis |
| `stakeholder-input/` | Input dan kebutuhan dari stakeholder |

## Cara Penggunaan

Setiap diskusi disimpan sebagai file `.md` dengan format nama:
`YYYY-MM-DD-topik-diskusi.md`
